    1. 매 주 해당 주차의 **각 강의마다 최소 1개의 질문거리**를 갖고 올 것 (해당 강의 발표자 제외)

       - **강의 중 이해가 잘 되지 않는 부분**

       - 위 질문거리를 준비하지 않았을 시, 기존 과제 미제출 상황과 동일한 벌금 부과
